# News

#### 项目介绍
微信小程序项目
涉及功能

1.仿wx.showActionSheet组件， 因为wx.showActionSheet组件不可以调用分享，so自己手动实现

2.canvas文章内容自动换行，内边距功能实现

3.canvas动态生成图片功能

4.文章左右滑动查看功能
